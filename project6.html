For W.E.B., the diploma thesis “AI model for form evaluation” delivered a practical AI system that automatically analyzes documents such as PDFs, scanned forms, or images and extracts relevant information in a structured format – fully local and without cloud dependencies.

End-to-End Workflow

  • Document intake: Upload PDFs, images, or DOCX; files are stored and preprocessed in the system.
  • OCR text recognition: Tesseract converts scans to text, including cleanup and structuring.
  • AI processing (LLM): A local LLM detects form fields and values and returns structured JSON outputs.
  • Alternative pipeline: Donut processes documents directly as images to compare against OCR-based results.
  • Storage & status: Results are stored in the database, documents remain retrievable, and statuses update in real time.

My Contribution (AI Focus)

  • Model evaluation: Compared classic, multimodal, and document-specific open-source models for accuracy, compute cost, and JSON output quality. Result: Qwen 2.5 as the main model.
  • Prompt engineering: Designed robust prompts for reliable field detection and consistent machine-readable JSON even with OCR noise.
  • AI pipeline & logic: Prepared OCR text, validated model responses, and returned structured outputs.
  • Local deployment: Implemented a fully local AI stack with GPU acceleration (CUDA/cuDNN) and full data control.
