Event Dashboard

Das Event Dashboard bündelt Daten zu lokalen Meet-Ups und Workshops. Teilnehmer können sich anmelden, Feedback geben und vergangene Events nachverfolgen.

  • Technologien: TypeScript, SvelteKit, Supabase, Tailwind CSS
  • Features: Live-Teilnehmerzahlen, Session-Feedback in Echtzeit und automatische Zusammenfassungen
  • Lernfokus: Reaktive Frontends, Datenvisualisierung und Deployment in der Cloud
