Elster Crypto Scout is a semi-automated research setup that monitors the latest DEX Screener drops and paper trades promising meme coins with virtual capital. This lets us validate strategies under real conditions without risking real money.

Over our three-month sprint, we built a modular structure that can later support real transactions. My focus was on clean data intake, extensible analytics modules, and an operator dashboard that keeps experiments safe.

Screens from the Elster dashboard show the full trading flow from coin detection to evaluating each simulated position.

  • Team: 4 contributors
  • Focus: Automated trading playbook
  • Stage: Active prototype

Project Overview

The workflow listens to the DEX Screener API feed, scores brand-new tokens with custom KPIs, and automatically opens positions with play money. Stop-loss presets and manual exits simulate a full trading process and deliver telemetry for fast learnings.

Key Highlights

  • Live intake of coin drops with filtering logic that scores new launches against custom KPIs.
  • Scenario-based paper trading including stop-loss presets and automatic sell triggers.
  • Meaningful telemetry for each trade to speed up strategy reviews.

Tech Stack

  • DEX Screener API
  • C#
  • Blazor
  • SignalR
  • PostgreSQL

My Contributions

  • Built the frontend dashboards for holdings and historical trades.
  • Documented the workflow for later integrations with real capital.
