Automation Toolkit

Das Automation Toolkit bündelt mehrere Skripte, die meinen Alltag in Schule und Verein vereinfachen – vom automatischen Export von Stundenplänen bis hin zur Synchronisation von Meeting-Notizen.

  • Technologien: Python, PowerShell, GitHub Actions
  • Features: Datei-Backups, Kalender-Sync, Statusberichte per Mail
  • Nutzen: Spart pro Woche mehrere Stunden manueller Routinearbeit
