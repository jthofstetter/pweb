Portfolio CMS

Um Inhalte schneller austauschen zu können, habe ich ein kleines statisches CMS gebaut. Es generiert Seiten aus Markdown-Dateien und deployt per GitHub Actions.

  • Technologien: Node.js, Eleventy, Tailwind CSS
  • Features: Markdown-zu-HTML Build-Pipeline, automatisches Deployment, Komponentenbibliothek
  • Lernfokus: Content-Workflows, CI/CD und wiederverwendbare UI-Bausteine
