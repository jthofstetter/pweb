Superstruct is a 3D Unity game we built as a school project. The central elevator acts as a hub between multiple floors packed with minigames. In each section, players collect resources to power the elevator and unlock the next level.

Each floor presents its own scenario – from dexterity sequences to small puzzle rooms. That creates a loop of exploring, collecting, and upgrading that keeps players motivated to climb higher.

  • Team: 6 classmates
  • Focus: Elevator hub, resource progression, and minigame mix
  • Stage: Playable prototype with vertical slice

Project Overview

The code organizes floors as individual Unity scenes that the elevator loads at runtime. A resource manager tracks which items are earned in minigames and which upgrades those items can fund. The door to the next floor only opens once enough material is collected.

Core Gameplay Features

  • Central elevator as the navigation point with transition animations between floors.
  • Three minigame types (dexterity, reaction, resource puzzle) that unlock sequentially.
  • Progress display showing how many materials are still missing to reach the next level.

Tech Stack

  • Unity Engine (3D)
  • C# gameplay scripting
  • Unity Animator & Timeline for cutscenes

My Contributions

  • Developed two minigames with resource rewards and difficulty curve.
  • Created UI displays for progress, inventory, and objectives on each floor.
